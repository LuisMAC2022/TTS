Capítulo 2. Introducción.

DaVinci Resolve veintiuno presenta una nueva página llamada Photo.

Esta página permite que coloristas y fotógrafos usen, en fotografías fijas, las herramientas de color avanzadas asociadas al flujo de trabajo de Hollywood.

La página Photo permite importar y administrar fotografías, e integrarlas con la página Color para realizar gradación basada en nodos.

Puedes usar corrección primaria de color, curvas, qualifiers, power windows y el editor de nodos.

Con estas herramientas es posible ajustar el color de forma artística en todo un proyecto fotográfico.

También puedes usar un panel de control DaVinci para ajustar varios parámetros al mismo tiempo.

Otras actualizaciones incluyen nuevas herramientas de inteligencia artificial.

IntelliSearch permite buscar contenido con rapidez.

CineFocus permite ajustar el punto focal.

También se incluyen herramientas de refinamiento facial.

El conjunto Krokodove agrega más de setenta herramientas gráficas a Fusion.

Las carpetas de Fairlight simplifican la gestión de pistas de audio.

También hay mejoras en keyframes, pases de recorte MultiMaster y visualización de gráficos de nodos como listas de capas.

Esta guía presenta las nuevas funciones y se incorporará al manual de referencia de DaVinci Resolve después de la beta pública.

Capítulo 3. Photo.

Edición de imágenes fotográficas integrada completamente dentro de DaVinci Resolve.

Sección 3.1. Página Photo.

DaVinci Resolve veintiuno introduce una nueva y potente página Photo.

Esta página está diseñada para administrar, organizar y graduar proyectos fotográficos con las herramientas de color y gestión de color nativas de Resolve.

Es un editor de imágenes integrado con una canalización de edición de video.

La página Photo funciona principalmente como una aplicación de edición fotográfica.

Su integración con el resto de DaVinci Resolve abre posibilidades creativas que normalmente no existen en un editor de imágenes fijas.

Mediante Photo Albums puedes pasar imágenes fijas de resolución completa a las páginas Cut, Edit, Color y Fusion para aplicar procesamiento adicional.

La página Photo incluye gestión completa de imágenes: etiquetas, favoritos, calificaciones y colecciones en álbumes.

También puedes conectar una cámara directamente a la página Photo para capturar imágenes y controlar ajustes de cámara en vivo.

Con IntelliSearch, la inteligencia artificial analiza tus imágenes y permite buscarlas por contenido, objetos, personas, colores y otros criterios.

La página Photo incluye una biblioteca amplia de efectos, con varios efectos Resolve FX y Fusion FX listos para usar.

La incorporación de Photo permite moverse con fluidez entre fotografía, edición de video, gradación de color, efectos visuales y posproducción de audio.

Nota importante. Las gradaciones y efectos aplicados a imágenes fijas dentro de álbumes se aplican a nivel de fuente.

Esto significa que las gradaciones de una imagen se comparten entre todos los álbumes de un proyecto, de forma similar a las gradaciones remotas.

Sección 3.2. Uso de la página Photo.

En la parte superior de la página Photo hay una barra de herramientas.

Sus botones permiten mostrar u ocultar distintas partes de la interfaz.

Media Pool. Abre u oculta una versión reducida del Media Pool completo, con acceso a las imágenes usadas en el proyecto.

Effects Library. Abre u oculta el repositorio de transiciones, generadores y efectos disponibles para la página Photo.

Photo Album. Abre el álbum fotográfico situado en la parte inferior de la interfaz.

Quick Export. Abre el cuadro de exportación rápida para entregar fotos de forma sencilla.

Metadata. Muestra u oculta el editor de metadatos.

Capture. Muestra u oculta la interfaz para capturar fotografías conectando la cámara al equipo.

Inspector. Muestra u oculta el inspector, donde puedes editar parámetros RAW, parámetros de foto y propiedades de efectos.

Sección 3.3. Media Pool en Photo.

En la página Photo, el Media Pool contiene solamente los medios de imagen fija que has importado para editar en el proyecto.

También contiene los álbumes que usarás para organizar tus fotografías.

A diferencia de los Media Pools de otras páginas, aquí no se muestran video, audio, líneas de tiempo ni clips de efectos.

Esos elementos siguen existiendo, pero se ocultan porque la página Photo está dedicada a imágenes fijas.

Al pulsar la lista de bins en la esquina superior izquierda aparece una jerarquía de carpetas para organizar fotos y álbumes.

De forma predeterminada, el Media Pool contiene un solo bin llamado Master.

Puedes agregar más bins para organizar clips y líneas de tiempo desde la lista de bins, eligiendo Add Bin.

Puedes renombrar un bin personalizado haciendo doble clic sobre su nombre o usando la opción Rename Bin del menú contextual.

El navegador muestra el contenido de la carpeta seleccionada.

Cada foto fija, bin y álbum aparece en esta zona.

Puedes crear tantos álbumes como necesites dentro de un mismo proyecto.

El Media Pool puede mostrarse en vista de miniaturas o en vista de lista.

En vista de lista, puedes ordenar el contenido por una parte de los metadatos disponibles en el editor de metadatos de la página Media.

Sección 3.4. Importación de fotos en la página Photo.

Agregar fotos desde la página Media ofrece más flexibilidad organizativa.

Aun así, si estás en la página Photo y necesitas importar fotos rápidamente, hay varios métodos disponibles.

Para usar el icono Import Media, abre la página Photo y pulsa el icono de lista Import Media en la parte superior del Media Pool.

Import Media abre un selector de archivos para importar fotos seleccionadas.

Import Media Folder abre un selector de archivos para importar todas las fotos dentro de una carpeta.

Import from Photos, disponible solo en Mac OS, abre la aplicación Fotos de Mac OS para seleccionar imágenes.

Import from Lightroom importa un catálogo de Adobe Lightroom.

Import Blackmagic Cloud Folder permite vincular una carpeta de Blackmagic Cloud e importar las fotos almacenadas allí.

También puedes hacer clic derecho en el Media Pool y elegir Import Media.

Después, usa el diálogo de importación para seleccionar una o más fotos y pulsa Open.

Las fotos se agregan al Media Pool del proyecto.

Sección 3.5. Bins y Smart Bins.

Hay dos clases de bins en el Media Pool, y cada una aparece en su propia sección.

Los bins simples se llenan manualmente.

Puedes arrastrar elementos a un bin, y allí permanecerán hasta que los muevas a otro bin.

Los bins pueden organizarse jerárquicamente, de modo que puedes crear una estructura de carpetas anidadas.

Para crear un bin, haz clic derecho en la lista de bins y elige Add Bin.

Los Smart Bins se llenan mediante reglas.

Estas reglas usan metadatos para filtrar dinámicamente el contenido del Media Pool cuando seleccionas el Smart Bin.

Esto permite organizar proyectos rápidamente si tú o un asistente agregaron metadatos a las fotos.

Los metadatos pueden incluir datos EXIF, palabras clave, comentarios, descripciones y otra información útil.

Para crear un Smart Bin, abre la lista de bins y elige Add Smart Bin.

Aparece un cuadro de diálogo donde puedes editar el nombre del bin y las reglas de filtrado.

Sección 3.6. Búsqueda en el Media Pool.

El icono de lupa abre un campo de búsqueda para localizar nombres de imágenes específicos.

Además, IntelliSearch permite usar inteligencia artificial para búsquedas casi instantáneas de personas y objetos escritos en ese campo.

Sección 3.7. Biblioteca de efectos.

Todos los efectos que puedes agregar a tus fotos están en el panel Effects.

El panel muestra una lista jerárquica de efectos y filtros, ordenados por categoría.

Para previsualizar un efecto antes de aplicarlo, activa Hover Scrub Preview en el menú de opciones de Effects.

Después coloca el puntero sobre una miniatura del panel Effects y muévelo a través de ella.

El efecto se previsualiza en el Viewer con sus parámetros predeterminados.

Para activar un efecto sobre una foto, arrastra la miniatura del efecto seleccionado hasta una foto en el álbum.

Para ajustar los parámetros del efecto, abre la pestaña Effects en el Inspector.

Puedes buscar un efecto escribiendo en la barra de búsqueda de Effects Library.

La combinación Shift y espacio abre el cuadro Effects Search en la página Photo.

Si una foto está seleccionada, al pulsar el efecto en la lista de búsqueda se aplica a esa foto.

Puedes buscar efectos y transiciones usando nombres de categorías.

Por ejemplo, al buscar Stylize aparecen los Resolve FX incluidos en la sección Stylize.

También puedes buscar efectos en el idioma configurado en las preferencias de interfaz de DaVinci Resolve, además del inglés.

Sección 3.8. Photo Albums.

La pestaña Photo Album abre el álbum fotográfico en la parte inferior de la interfaz.

Como en un álbum físico, los Photo Albums sirven para guardar y organizar colecciones de imágenes fijas.

En DaVinci Resolve pueden entenderse como un tipo especial de línea de tiempo, construida con fotos seleccionadas por el usuario.

Organizar imágenes del Media Pool en álbumes es esencial para aprovechar las capacidades fotográficas de DaVinci Resolve veintiuno.

Las fotos son independientes de resolución y velocidad de fotogramas, mientras que las líneas de tiempo no lo son.

La página Photo usa el Photo Album como contenedor organizativo central.

Esto permite que las fotos interactúen con el resto de páginas de DaVinci Resolve.

Para crear un Photo Album, haz clic derecho en el Media Pool y elige New Photo Album.

También puedes ir al menú File y elegir New Photo Album.

Para agregar imágenes a un álbum, selecciona una foto o un grupo de fotos en el Media Pool.

Después haz clic derecho en una imagen y elige Add Photo o Add Photos to Album.

Las fotos seleccionadas se agregan al álbum actual.

Para eliminar imágenes de un álbum, selecciona una o varias fotos y elige Delete Selected.

Las fotos se eliminan del álbum actual, sin necesidad de borrar el archivo original del proyecto.

Después de crear y llenar un álbum, puedes usar sus herramientas para organizar, ordenar y etiquetar imágenes fijas.

Puedes agregar etiquetas, colores y flags a imágenes en el Viewer.

Luego puedes usar los filtros del Photo Album para organizarlas y clasificarlas.

Las miniaturas del álbum muestran iconos que indican operaciones aplicadas a la foto.

Un rectángulo arcoíris alrededor del número indica que se aplicó una operación RAW o una operación de Photo.

La calificación por estrellas muestra la puntuación asignada por el usuario, y puede modificarse con un clic.

El flag en la esquina superior derecha indica qué bandera de color se asignó a la foto.

La cápsula pequeña en la esquina inferior derecha indica el color asignado.

Un corazón o una equis muestra si la foto fue marcada como favorita o rechazada.

El formato de imagen aparece bajo la miniatura, en la esquina inferior izquierda.

El indicador FX muestra que se aplicó un efecto.

El icono Transform indica que se aplicó una transformación.

El icono Crop indica que se aplicó un recorte.

Sección 3.9. Filtros del Photo Album.

Tags filtra las fotos según etiquetas.

Con clic derecho puedes elegir criterios como Good Take, Untagged, Rejected o una combinación de ellos.

Rating filtra por calificación.

Con clic derecho puedes elegir igual a, menor o igual, mayor o igual, y la cantidad exacta de estrellas.

Flags filtra según banderas de color.

Con clic derecho puedes elegir uno o varios colores como criterio.

Color filtra por color de clip.

Keywords filtra por palabras clave.

Sort Photos cambia el orden ascendente o descendente.

Con clic derecho puedes elegir el valor de metadatos que se usará para ordenar.

Para filtros más avanzados, usa el menú desplegable junto a la etiqueta del Photo Album.

Puedes filtrar por atributos como fotos graduadas o no graduadas, personas, o fotos modificadas en las últimas horas.

Los Photo Albums pueden trasladar fotos a resolución nativa entre páginas de DaVinci Resolve.

Esto permite usar herramientas de Cut, Edit, Color y Fusion sobre imágenes fijas, además de la página Photo.

Sección 3.10. Viewer de Photo.

El Viewer de Photo permite observar con detalle una foto específica durante la edición.

También contiene herramientas propias para trabajar con las imágenes.

En la barra superior, Photo Album muestra la imagen seleccionada.

Capture Live View abre la vista en vivo cuando se trabaja con cámara conectada.

Zoom Percent permite elegir el nivel de zoom del Viewer.

Album List muestra el álbum actual y permite abrir álbumes recientes desde un menú desplegable.

Single View abre un visor grande de una sola imagen.

Lightbox muestra miniaturas redimensionables de todas las imágenes del álbum.

Viewer Guides abre guías para relaciones de aspecto, áreas seguras, reglas personalizadas y líneas de guía.

Bypass Color and Fusion omite efectos de Color o Fusion que se hayan aplicado.

Puedes elegir omitir solo Color o solo Fusion desde el menú desplegable.

Sección 3.11. Lightbox y modo mano.

Al pulsar Lightbox, la vista de una sola foto se reemplaza por miniaturas redimensionables de cada foto del álbum.

Esto permite ver varias fotos con mayor tamaño y obtener una visión general del contenido del álbum.

También puedes seleccionar una imagen y graduarla en vivo mientras ves el resultado actualizado en toda la colección.

Mantener presionada la tecla H en el Viewer activa el modo mano.

Cuando el modo mano está activo, la imagen se escala al cien por ciento.

Arrastrar la mano dentro del Viewer desplaza la imagen.

Un doble clic sobre un área permite examinar píxeles individuales.

Otro doble clic ajusta la imagen al Viewer.

En la barra inferior, Tools abre las interfaces de Transform, Crop y Effects Overlay.

Exif Data muestra ISO, velocidad de obturación, apertura y metadatos de lente.

Favorite marca la imagen como favorita con un pequeño corazón.

Untag elimina la marca de favorita o rechazada.

Reject marca la imagen como rechazada con una pequeña equis.

Version Selector permite elegir otra versión guardada de la imagen.

Rating asigna una calificación de una a cinco estrellas.

Color asigna una etiqueta de color útil para operaciones organizativas en otras páginas.

Flag asigna una bandera de color útil para operaciones organizativas en otras páginas.

Sección 3.12. Transform.

Al seleccionar Transform aparecen controles en pantalla para manipular directamente la imagen en el Viewer.

Puedes arrastrar dentro del cuadro delimitador del clip para ajustar paneo e inclinación.

Puedes arrastrar una esquina diagonal para redimensionar proporcionalmente.

También puedes arrastrar lados superiores, inferiores, izquierdos o derechos para comprimir o estirar la anchura o la altura.

El tirador central permite rotar la imagen.

Mientras se usan controles en pantalla, la herramienta muestra siempre el tamaño original de la foto bajo la cuadrícula de transformación.

Esto permite ver zonas fuera del encuadre mientras haces ajustes.

Al cerrar la barra, la operación de transformación se activa y se aplica a la imagen.

El botón Toggle activa o desactiva un efecto sin perder los ajustes personalizados.

Zoom Width y Height permiten agrandar o reducir la imagen.

Los parámetros X y Y pueden vincularse para bloquear la relación de aspecto.

También pueden separarse para estirar o comprimir la imagen en una sola dirección.

Zoom Link activa el bloqueo para unir ancho y alto durante una operación de zoom.

Position X y Y mueve la imagen dentro del encuadre.

X mueve la imagen a izquierda o derecha.

Y mueve la imagen hacia arriba o hacia abajo.

Rotation Angle gira la imagen alrededor del punto de anclaje.

Pitch gira la imagen hacia la cámara o lejos de ella sobre un eje horizontal que atraviesa el centro.

Valores positivos empujan la parte superior hacia atrás y acercan la parte inferior.

Valores negativos acercan la parte superior y empujan la parte inferior hacia atrás.

Yaw gira la imagen sobre un eje vertical que atraviesa el centro.

Valores positivos acercan el lado izquierdo y alejan el derecho.

Valores negativos alejan el lado izquierdo y acercan el derecho.

Flip Horizontal invierte la imagen sobre el eje X, de izquierda a derecha.

Flip Vertical invierte el clip sobre el eje Y, dejándolo de cabeza.

Reset devuelve los parámetros de una categoría a sus valores predeterminados.

Sección 3.13. Crop.

La página Photo incluye controles en pantalla para recortar directamente la imagen en el Viewer.

Cada lado de la imagen tiene un tirador individual para recortar ese lado.

Estos parámetros también se editan desde el Video Inspector.

Mientras se usan controles en pantalla, la herramienta muestra el tamaño original de la foto bajo la cuadrícula de recorte.

Esto permite ver zonas fuera del encuadre al hacer ajustes.

Al cerrar la barra, el recorte se activa y se aplica a la imagen.

Aspect Ratio permite elegir relaciones de aspecto comunes para formatos de foto y video, o crear una relación personalizada.

El valor predeterminado es As Shot, que corresponde a la relación nativa de la cámara.

Lock Aspect Ratio bloquea la relación de aspecto seleccionada durante los recortes.

Si está desbloqueado, puedes recortar en forma libre.

Crop Left, Right, Top y Bottom cortan los cuatro lados de la imagen en píxeles.

Reset devuelve los controles de recorte a sus valores predeterminados.

Sección 3.14. Overlays de Open FX y Fusion.

Si aplicaste un efecto Open FX o Fusion que tiene controles en pantalla, puedes acceder a ellos desde los iconos correspondientes.

Sección 3.15. Versiones en Photo Album.

Una de las funciones más útiles de Photo Album es el versionado.

Puedes ajustar una foto, guardar esos cambios como versión y luego probar un aspecto radicalmente distinto sobre la misma imagen.

La versión original permanece guardada.

Puedes guardar tantas versiones como necesites.

Para crear una versión, realiza tus ediciones, haz clic derecho en una foto del Photo Album y elige New Version.

El cuadro New Version permite escribir un nombre descriptivo y confirmar con OK.

Cambiar de versión también es sencillo.

Las versiones guardadas aparecen en el mismo menú contextual donde fueron creadas.

También hay un selector desplegable en la parte inferior del Photo Viewer.

Importante. Solo la versión seleccionada se exporta en Quick Export y en la página Deliver.

Las demás versiones se ignoran al exportar.

Sección 3.16. Quick Export.

Quick Export abre una ventana de entrega diseñada para imágenes fijas.

Seleccionas el formato de imagen a la izquierda, introduces los parámetros a la derecha y pulsas Export.

Los parámetros exactos dependen del formato elegido.

Export permite exportar todo el álbum o solo las fotos seleccionadas.

File Name permite agregar texto como prefijo de nombre de archivo.

Scale permite elegir resolución original o una reducción porcentual para archivos más pequeños.

Include Metadata permite incluir los metadatos de la foto en la imagen final.

Consejo. Si necesitas más control sobre la exportación de fotos, usa las herramientas de la página Deliver.

Sección 3.17. Metadata.

En la página Photo, el editor de metadatos se abre en el mismo lugar que el Inspector.

Cuando seleccionas una foto en el Media Pool o en el álbum, sus metadatos aparecen en el editor.

Si seleccionas varios clips, solo se muestra la información del primero.

El encabezado del editor contiene información no editable: nombre de archivo, directorio, duración, velocidad de fotogramas, resolución y códec.

Hay muchos campos de metadatos disponibles.

Por eso, dos menús desplegables superiores cambian el conjunto de metadatos mostrado.

Metadata Presets permite elegir conjuntos personalizados creados en preferencias de usuario.

Metadata Groups permite cambiar entre grupos de metadatos organizados para tareas o flujos de trabajo específicos.

La parte central del editor contiene campos editables para revisar y modificar distintos criterios de metadatos.

Sección 3.18. Capture.

Capture abre la interfaz Capture Live View para disparo conectado desde la cámara directamente hacia DaVinci Resolve.

Permite transferir fotos de inmediato al Media Pool o a un Photo Album.

También sirve para edición, revisión con clientes o evaluación de la imagen en una pantalla más grande.

Primero configura Camera Tethering en la cámara.

Cada marca y modelo tiene métodos distintos, así que conviene consultar el manual de la cámara.

Después conecta la cámara a la estación de trabajo con un cable USB.

Como nota de beta, solo las cámaras Canon y Sony son compatibles con captura conectada en vivo.

En Mac OS, Canon y Sony son compatibles de forma nativa.

En Windows, Canon es compatible de forma nativa.

Sony es compatible, pero requiere instalar software adicional de Sony Creators Cloud.

En Linux no se admite tethering.

En iPad tampoco se admite tethering.

Si hay problemas de conexión, asegúrate de que no haya otros programas abiertos que se conecten automáticamente a la cámara.

A veces es necesario reiniciar DaVinci Resolve para conectar la cámara.

Cuando la cámara y el equipo están configurados, al pulsar Capture se conecta automáticamente con la cámara.

Al pulsar Capture Live View en la parte superior del Viewer se abre la interfaz de vista en vivo.

En la columna derecha aparecen controles de captura.

Según el modelo de cámara, podrás cambiar ajustes directamente desde la interfaz.

Se pueden controlar parámetros como apertura, velocidad de obturación, ISO y compensación de exposición.

Debajo puedes ajustar el foco girando la rueda o pulsando botones de dirección.

Más abajo, en Settings, puedes ajustar opciones avanzadas.

Los ajustes no admitidos por la cámara aparecen desactivados.

A la izquierda aparece Capture Live View, que se actualiza en tiempo real con lo que ve la cámara.

Puede incluir visualizaciones de pantalla como cajas de enfoque y focus peaking.

En el extremo izquierdo aparece la última foto capturada.

Así puedes compararla con la configuración actual de la toma.

Puedes pausar Live View en cualquier momento usando el interruptor de Capture Live View.

Cuando la cámara muestra la imagen deseada, captura la foto pulsando el disparador en la esquina superior derecha de los controles.

Camera Name muestra el modelo de cámara conectado.

Half Press envía a la cámara una señal de media pulsación, usada a menudo para bloquear el autofoco.

Shutter Release activa el disparador de la cámara y toma una foto.

Aperture muestra la apertura seleccionada del lente.

Shutter Speed muestra la velocidad de obturación actual.

ISO muestra el ISO seleccionado.

Exposure Compensation muestra la compensación de exposición actual.

AutoFocus activa el autofoco.

Control Wheel ajusta el parámetro activo y controlable.

Enable Capture Live Preview activa la previsualización de cámara en vivo dentro del Viewer.

Los scopes de Capture son idénticos a los del Inspector y se actualizan en tiempo real desde la cámara.

Settings permite controlar la cámara conectada desde la interfaz.

Shooting Mode define el modo de disparo, como prioridad de apertura, prioridad de obturación, programa o manual.

Aperture ajusta la apertura del lente.

Shutter ajusta la velocidad de obturación.

ISO ajusta el ISO de la cámara.

Exposure Comp define los pasos de compensación de exposición.

White Balance define el preset de balance de blancos.

Temperature define una temperatura exacta de balance de blancos en grados Kelvin.

Drive define el modo de disparo de la cámara, como temporizador o alta velocidad.

Focus Mode define el modo de enfoque, por ejemplo AF-S, AF-C o Manual.

Focus Area define el área de enfoque automático, por ejemplo Wide, Center o Zone.

File Settings define el formato de captura, por ejemplo RAW, JPEG, o RAW más JPEG.

File Size define el tamaño de archivo de la foto.

Sección 3.21. RAW.

Las herramientas RAW editan la imagen con datos RAW anteriores al debayer.

La edición RAW es no destructiva y suele estar ajustada a la cámara específica.

Por eso es el método preferido para modificar exposición y color.

Si el archivo no es RAW, esta pestaña aparece desactivada.

Decode Quality permite debayerizar archivos RAW a resolución completa, media o de un cuarto.

Esto mejora el rendimiento en sistemas más lentos.

La menor resolución se procesa más rápido, aunque con menor calidad de previsualización.

Puedes trabajar con baja resolución y cambiar a mayor calidad al renderizar la salida final.

Project Settings permite usar la calidad de decodificación definida para todo el proyecto.

White Balance usa el balance de blancos capturado en cámara o presets como Daylight, Cloudy, Shade, Tungsten, Fluorescent y Flash.

Color Space define el espacio de color para editar y entregar la foto.

En RAW suele convenir usar un espacio de color del fabricante de la cámara o un estándar de video común.

Gamma define la curva gamma aplicada a la imagen.

Puedes elegir una curva del fabricante de la cámara o gammas de estándares de video.

Temperature ajusta la temperatura de color del balance de blancos.

Valores bajos corrigen iluminación cálida.

Valores altos corrigen iluminación fría.

Tint modifica el equilibrio entre verde y magenta, útil para tintes fluorescentes.

Exposure sube o baja la luminosidad en unidades relativas a pasos de exposición.

Aunque el ajuste eleve datos por encima del blanco máximo, la información se conserva para ajustes posteriores.

Sharpness aplica un filtro de nitidez específico del debayer para aumentar la apariencia de detalle.

Highlights recupera detalle en altas luces quemadas en material de alto rango dinámico.

Shadows aclara u oscurece selectivamente el detalle de sombras.

Color Boost aumenta de forma natural la saturación en regiones de baja saturación, como una operación de vibrance.

Saturation ajusta la intensidad global del color.

Midtone Detail aumenta el contraste de zonas con detalle de borde para mejorar la percepción de nitidez.

Con valores negativos suaviza zonas de bajo detalle sin afectar tanto las zonas detalladas.

Lift ajusta el punto negro del material, subiéndolo o bajándolo y escalando los medios tonos.

Gain ajusta el punto blanco, subiéndolo o bajándolo y escalando los medios tonos entre negro y blanco.

Contrast aumenta o reduce sombras y altas luces, dejando los medios tonos alrededor del cincuenta por ciento relativamente estables.

Sección 3.25. Ejemplo de edición fotográfica.

Combinar un editor de imagen fija con las herramientas de cine y video de DaVinci Resolve habilita efectos fotográficos creativos y prácticos.

Por ejemplo, puedes tomar una foto común de teléfono y hacer que parezca capturada en película cinematográfica de treinta y cinco milímetros.

Primero usa el Photo Inspector para hacer ajustes estándar de exposición y balance de color.

En el ejemplo, el sujeto de unas estatuas se confunde con el fondo.

La foto fue tomada con un lente pequeño de teléfono, de modo que todo, incluido el fondo, aparece en foco.

Esto hace que el fondo y las estatuas se mezclen visualmente.

Para solucionarlo, se arrastra el nuevo efecto CineFocus sobre la imagen del álbum.

CineFocus crea un lente virtual nuevo para observar la imagen.

El lente puede tener mayor apertura, menor profundidad de campo y un bokeh más agradable.

CineFocus analiza primero la foto para crear un mapa de profundidad de distancias percibidas dentro de una imagen bidimensional.

Luego, con Focus mode, puedes definir la apertura exacta y la profundidad de campo que aíslan las estatuas en primer plano.

El fondo queda fuera de foco.

Después puedes ajustar parámetros del lente, desde una simulación básica hasta un lente anamórfico de once hojas.

También puedes agregar aberración cromática o curvatura de lente ligeramente descentrada.

En el ejemplo se mantienen los parámetros predeterminados.

La imagen resultante muestra el fondo desenfocado detrás de las estatuas.

El desenfoque aumenta gradualmente con la distancia, lo cual es más realista que enmascarar un área y desenfocarla sin profundidad.

Para completar el efecto, se aplica Film Look Creator sobre la imagen en el Photo Album.

Así trabajan dos efectos sobre la misma imagen.

Film Look Creator permite ajustar halation, grano, bloom, viñeteado y otros parámetros de película.

En el ejemplo se aplica el preset Default treinta y cinco milímetros.

Esto agrega un perfil de color de película y un patrón de grano con tamaño adecuado.

El resultado final separa claramente las estatuas del fondo y agrega un aspecto fílmico al encuadre.

Sección 4.1. AI IntelliSearch.

DaVinci Resolve veintiuno incorpora IntelliSearch, una herramienta avanzada de organización de medios.

IntelliSearch usa inteligencia artificial para ofrecer búsquedas completas y casi instantáneas de personas y objetos.

La mejora de eficiencia al encontrar medios específicos es inmediata.

Cuando recuerdes que había un clip de algo pero no sepas dónde está, puedes pedirlo a DaVinci Resolve y el programa lo encontrará.

Como operación basada en inteligencia artificial, IntelliSearch requiere descargar un modelo desde Extras Download Manager.

Una vez instalado, debes analizar los medios del Media Pool antes de usar IntelliSearch desde la barra de búsqueda.

Hay dos modelos: Faster Mode y Better Mode.

Puedes descargar uno o ambos.

Tus modos disponibles dependen de los paquetes instalados.

Al inicio debes elegir si IntelliSearch funciona en Faster o Better.

La diferencia está en el tiempo de análisis y el tamaño de la descarga del modelo.

Ambos modelos son igual de rápidos después del análisis.

Better tarda más en analizar clips, pero ofrece resultados más precisos.

Puedes comenzar con Faster y ejecutar Better después cuando la estación de trabajo tenga más tiempo libre.

La selección se hace desde el menú de tres puntos del Media Pool, en IntelliSearch Mode.

Para analizar medios, usa AI Clip Analysis.

Esta herramienta aplica distintos modelos de análisis de inteligencia artificial a clips del Media Pool.

Por defecto analiza todos los clips del Media Pool.

Con el menú desplegable puedes analizar solo clips seleccionados.

Para iniciar el análisis, activa Analyze for IntelliSearch.

La casilla Identify Faces permite reconocer humanos individuales.

Así puedes agregar nombres como criterios de búsqueda en Manage Faces.

Al pulsar Analyze aparece una ventana de progreso con una estimación de finalización.

Cuando termina, los clips analizados muestran el icono AI Analysis en la miniatura.

IntelliSearch se abre con el icono de lupa en el Media Pool.

Search Parameters define qué tipo de análisis se aplica a los términos.

Search All devuelve resultados de todos los análisis de inteligencia artificial y de los metadatos introducidos.

Search Visual Only devuelve resultados basados solo en análisis de imagen.

La búsqueda visual permite localizar objetos y contenido dentro de la escena.

Search Transcript Only devuelve resultados del audio transcrito.

Es útil para buscar frases exactas.

Metadata Search devuelve resultados de campos de metadatos.

También permite filtrar por palabras clave, propiedades de clips y metadatos personalizados.

Search Field es el campo donde escribes lo que quieres encontrar.

Puede ser un objeto, una persona, un color, un número de cámara o una línea de diálogo.

Add as SmartBin crea un Smart Bin persistente con los criterios de búsqueda seleccionados.

Show Face Gallery muestra miniaturas de cada rostro humano detectado.

Puedes pulsar un rostro para ver tomas donde aparece esa persona o limitar la búsqueda a ese rostro.

Manage Faces permite nombrar y curar los rostros detectados.

Search In elige dónde buscar: All Bins o Selected Bins.

Display define cómo se muestran los resultados.

Full Clips devuelve el clip completo como coincidencia y marca con una línea amarilla la duración relevante.

Sección 4.2. AI Speech Generator.

Speech Generator genera voz hablada de alta calidad a partir de texto escrito.

Conviene mantenerlo breve o dividir textos largos en piezas pequeñas.

Text Input es donde escribes el texto que debe leerse.

Los puntos generan pausas más largas.

Las comas producen pausas más cortas.

Los saltos de línea pueden alargar la pausa cuando siguen a un punto.

Valores positivos suben el tono.

Valores negativos lo bajan.

Generation ID funciona como una toma de interpretación.

Cada número representa una lectura con prosodia distinta.

El mismo número produce la misma interpretación.

Esto permite conservar énfasis emocional mientras pruebas otras variaciones.

Randomize usa un número aleatorio para Generation ID.

File Name define el nombre del archivo de audio.

El sistema agrega un sufijo numérico automáticamente.

Audio Track elige la pista de audio de la línea de tiempo.

La grabación debe estar limpia y con buena fidelidad.

Speech Generator usa solo los primeros veinte segundos si el archivo es más largo.

La voz debe estar bien nivelada y con poco ruido.

Conviene evitar silencios largos, cambios extremos de volumen o emoción si se busca un resultado uniforme.

Speech Generator crea audio hablado desde texto.

Voice Convert, en cambio, transforma una interpretación ya grabada para que suene con otra voz.
